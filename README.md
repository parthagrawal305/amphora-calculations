# Amphora Ads ROI Calculator

An interactive calculator to help publishers estimate their earnings from native ads that offset AI inference costs.

## Features

- Real-time calculations as you adjust inputs
- Per-conversation metrics breakdown
- Monthly projections
- Support for both CPM and CPC ad models
- Fill rate and CTR calculations

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`)

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
